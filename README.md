# ClearMind PDF - PDF to Structured Notes

Convert PDF documents into structured, AI-ready markdown notes with ease.

## Features

- 📄 PDF Upload & Processing
- 📝 Automatic Markdown Conversion
- 🏷️ Smart Section Detection
- 📊 Table Extraction
- 🔗 Reference Link Preservation
- 🎯 AI-Ready Output Format

## Tech Stack

- **Frontend**: Next.js 15 + TypeScript + Tailwind CSS
- **Backend**: Python FastAPI
- **PDF Processing**: Custom extraction pipeline

## Getting Started

### Prerequisites

- Node.js 18+
- Python 3.9+

### Installation

```bash
# Install frontend dependencies
npm install

# Install backend dependencies
cd backend
pip install -r requirements.txt
```

### Development

```bash
# Start frontend
npm run dev

# Start backend (in another terminal)
cd backend
python main.py
```

## Usage

1. Upload a PDF file
2. Wait for processing
3. Download the structured markdown output
